## ✨ Features

- **Premium Aesthetics**: High-end luxury design with soft shadows, glassmorphism, and large typography inspired by top-tier modern web practices.
- **Buttery Smooth Animations**: Integrated with **Framer Motion** and **Lenis** for flawless scroll effects, page transitions, and subtle micro-interactions.
- **Responsive Layout**: A fully responsive design ensuring a beautiful experience across desktop, tablet, and mobile devices.
- **Blazing Fast Performance**: Built on the Next.js App Router for server-side rendering, optimal SEO, and a Lighthouse score targeted above 95.
- **Bento Grids**: Dynamic and engaging layouts for clinic highlights and features.
- **SEO Ready**: Configured for high visibility with optimized metadata and semantic structure.

## 🛠 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [Lenis Smooth Scroll](https://lenis.studiofreight.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

First, clone the repository and install the dependencies:

```bash
git clone https://github.com/Abhi-Wathore/dental-clinic.git
cd dental-clinic
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## 📂 Project Structure

- `src/app/`: Next.js App Router pages and global layouts.
- `src/components/`: Reusable, animated UI components (Hero, About, Services, FAQ, etc.).
- `public/`: Static assets and icons.
